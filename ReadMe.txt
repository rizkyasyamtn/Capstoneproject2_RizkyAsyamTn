Analisis Data Penjualan SaaS 
Analisis Komprehensif Penjualan, Laba, dan Diskon

Masalah Bisnis & Tujuan Analisis:
Konteks:
Perusahaan SaaS ingin mendapatkan wawasan tentang performa penjualan, profitabilitas, dan strategi diskon.
Tujuan:
Menganalisis data penjualan untuk memahami performa regional, profitabilitas produk, serta dampak diskon terhadap penjualan dan laba.

Point analisis:
Gambaran Umum Dataset
Data Cleaning
Eksplorasi Data (EDA)
Distribusi Penjualan
Penjualan Berdasarkan Wilayah
Laba Berdasarkan Wilayah
Produk Teratas Berdasarkan Penjualan
Produk Teratas Berdasarkan Laba
Analisis Diskon
Analisis Regresi
